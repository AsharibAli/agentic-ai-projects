A simple chatbot API build with
- FastAPI
- OpenAI Agents SDK

## Setup

# Initialize the project
```bash
uv init fastapi-oas
```

# Add dependencies
```bash
uv add fastapi[standard] openai-agents pytest pytest-asyncio
```

# Activate the virtual environment
```bash
.venv/Scripts/activate
```

# Run the project
```bash
fastapi dev main.py
```
